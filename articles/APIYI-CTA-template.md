# APIYI 文章 CTA 标准模板

## 文章开头 CTA
在文章开始部分（通常在第一个主要段落之后）插入：

```markdown
欢迎免费试用我们的 API 服务产品：API易 https://www.apiyi.com/register/?aff_code=HDnF
```

## 文章结尾 CTA
在文章结束部分（通常在总结的前一段）插入：

```markdown
欢迎免费试用我们的 API 服务产品：API易 https://www.apiyi.com/register/?aff_code=HDnF

[![CTA：免费试用 API易](../images/APIYI-banner-2407.png)](https://www.apiyi.com/register/?aff_code=HDnF)
```

## CTA 使用规范

### 1. 位置规范
- 开头 CTA：放在文章开头第一个主要内容段落后
- 结尾 CTA：放在文章主体内容结束前，包含文字链接和图片 banner

### 2. 格式规范
- 使用标准推广链接，包含 aff_code
- Banner 图片使用相对路径
- 保持链接格式统一

### 3. 文案规范
- 保持友好专业的语气
- 强调"免费试用"
- 使用"我们的 API 服务产品"表述

### 4. 图片规范
- 使用标准 banner 图片：`APIYI-banner-2407.png`
- 图片放置在 `articles/images/` 目录下
- 保持图片链接格式统一

## 注意事项
1. 确保每篇文章都包含开头和结尾两处 CTA
2. 检查链接的有效性
3. 确保图片路径正确
4. 保持格式的一致性
